\# 🤖 QQ机器人 - XP12



一个功能丰富的 QQ 机器人，基于 LLOneBot 和 Ollama 实现，支持点歌、婚姻系统、AI聊天、防撤回等多种功能。

\[!\[GitHub license](https://img.shields.io/github/license/xp12-mcmin/qq-robot)](https://github.com/xp12-mcmin/qq-robot/blob/main/LICENSE)

\## 📋 功能列表



\### 🎵 点歌功能

\- 支持 QQ 音乐搜索

\- 自动下载音频文件

\- 转换为 QQ 语音消息（AMR格式）

\- 发送歌曲封面图和下载链接

\- 多源自动切换（QQ音乐、网易云等）



\### 💑 婚姻系统

\- 今日老婆（每日随机抽取群成员）

\- 结婚/离婚（双向绑定）

\- 夫妻排行榜

\- 结婚证图片生成



\### 🎭 AI 性格系统

\- 支持多种性格切换（默认助手、猫娘等）

\- 每个群可独立设置性格

\- 敏感词过滤（涉黄自动拦截）

\- 辱骂机器人检测（自动拉黑）

\- 违规次数累计，5次自动拉黑



\### 🛡️ 防撤回系统

\- 消息撤回保护

\- 自动重发被撤回的消息

\- 保护特定用户/群



\### ❤️ 好感度系统

\- 每日签到

\- 好感度商店

\- 称号系统

\- 好感度排行榜



\### 📅 自动打卡

\- 每日 00:00 自动打卡

\- 支持多群批量打卡

\- 手动打卡命令



\### 🎋 抽签系统

\- 每日运势签

\- 财运/姻缘/事业/学业签

\- 自动生成运势图片

\- 每天结果固定（次日重置）



\### 🎉 入群欢迎

\- 自定义欢迎语

\- 支持变量 `{name}`, `{user\_id}`, `{group\_id}`

\- 每个群独立设置



\### 🔍 联网搜索

\- 联网搜索信息

\- AI 联网问答



\### 📝 改名功能

\- 修改群成员名片（管理员）

\- 修改机器人自己的名片



\### 其他功能

\- AI 聊天（支持对话记忆）

\- 黑名单管理

\- 阴阳库

\- 群管理（禁言、解禁）

\- 刷屏检测

\- 自动解禁

\- 视频解析

\## 🚀 安装指南



\### 环境要求

\- Python 3.10+

\- Ollama（用于AI聊天）

\- LLOneBot（QQ协议适配）

\- ffmpeg（用于语音转换，可选）



\### 1. 克隆项目

```bash

git clone https://github.com/xp12-mcmin/qq-robot.git

cd qq-robot

```



\### 2. 安装依赖

```bash

pip install -r requirements.txt

```



核心依赖：

```bash

pip install websockets aiohttp flask requests pillow

```



\### 3. 安装 ffmpeg（用于语音转换，可选）



\*\*Windows:\*\*

\- 下载：https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-essentials.zip

\- 解压后将 `bin/ffmpeg.exe` 复制到项目目录



\*\*Linux:\*\*

```bash

sudo apt install ffmpeg

```



\*\*macOS:\*\*

```bash

brew install ffmpeg

```



\### 4. 安装 Ollama

```bash

\# Windows/Linux/macOS

curl -fsSL https://ollama.com/install.sh | sh

```



拉取模型：

```bash

ollama pull qwen2.5:3b

```



\### 5. 配置 LLOneBot

\- WebSocket 端口：`8765`

\- HTTP API 端口：`3000`

\- 启用 WebSocket 服务



\### 6. 运行机器人

```bash

python qai主程序.py

```



\### 7. 可选：独立点歌模块

```bash

python music\_bot.py

```

\## ⚙️ 配置说明



\### 主程序配置（`qai主程序.py`）



```python

\# AI模型配置

self.text\_models\_priority = \["qwen2.5:3b", "qwen2.5:1.5b"]

self.image\_model = "qwen2.5:3b"



\# 处罚配置

self.MAX\_VIOLATIONS = 5        # 最大违规次数

self.RESET\_TIME = 1800         # 重置时间（秒）

```



\### LLOneBot 配置

\- WebSocket 地址：`ws://127.0.0.1:8765`

\- HTTP API：`http://127.0.0.1:3000`

\- WebHook 地址：`http://127.0.0.1:8766`
